# HITSchedule

Android版工大课表app。[下载网页](http://hitschedule.github.io/)

#### 已实现

- [x] 自定义背景
- [x] 从vpn加载课表
- [ ] 从校园网加载课表
- [x] 手动添加删除课表
- [x] 修改已有课表项
- [x] 空教室查询
- [x] 校历查询
- [x] 校车查询
- [x] 滑动切换周数
- [x] 多语言切换

#### 待实现

- 完善课表解析函数
- 将课表导出为ICS文件
