# HITSchedule

Android版工大课表app。[下载网页](http://hitschedule.github.io/)

#### 已实现

- [x] 自定义背景
- [ ] 从微信平台加载课表 (暂时停用)
- [x] 从[本科教学管理与服务平台](http://jwts.hit.edu.cn)加载本科生课表
- [ ] 加载研究生课表 (暂未适配)
- [x] 通过[统一身份认证](https://ids.hit.edu.cn/authserver/login)验证身份
- [x] 手动添加删除课表
- [x] 修改已有课表项
- [ ] 空教室查询
- [ ] 校历查询
- [ ] 校车查询
- [x] 滑动切换周数
- [x] 多语言切换
- [x] 将课表导出为ICS文件
- [x] 从vpn加载课表
- [x] 从校园网加载课表
